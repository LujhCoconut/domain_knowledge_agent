# OS Security & Program Analysis

操作系统安全机制设计与程序分析，涵盖 MAC 访问控制、内存沙箱、动态追踪、策略提取和移动端分析。

## 子目录

| 目录 | 主题 | 适合归档的内容 |
|------|------|----------------|
| `os-security/` | OS 安全 | 强制访问控制(MAC)、内存沙箱(SFI)、动态追踪(MPK/DBI)、策略提取、移动端安全分析 |

## 新增 skill 建议

- 按安全机制类型或分析平台命名，例如 `hardware-enclaves/`、`fuzzing/`、`binary-analysis/`。
- 每个 skill 应包含"问题 → 机制原理 → 工程实践 → 安全边界"的结构。
- 安全方向与 `performance/`、`architecture/` 有交叉，可相互引用。
