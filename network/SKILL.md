# Network Systems

网络系统设计与优化，涵盖用户态网络 runtime、内核网络栈、RPC 调度、去中心化架构。

## 子目录

| 目录 | 主题 | 适合归档的内容 |
|------|------|----------------|
| `os-networking/` | OS 网络 | 内核网络栈、用户态 runtime、TCP/RPC 调度、去中心化网络架构、消息语义优化 |

## 新增 skill 建议

- 按网络层级或范式命名，例如 `rdma/`、`http3-quic/`、`service-mesh/`。
- 每篇论文的知识归档到对应的 `KNOWLEDGE.md`，与 `performance/`、`security/` 保持一致。
